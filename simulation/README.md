## 시뮬레이션 / 구현
- 문제에서 말하는데로 잘 구현하는 것
- 문제를 읽고 제대로 이해하는 것이 매우 중요함
- 완전탐색과 함께 코테의 양대산맥
    - 실제로 완전탐색과 시뮬레이션 유형만 다 풀어도 합격하는 경우가 꽤 있음
- 프로그래밍 언어에 익숙하고 코딩을 많이 해봤다면 어느정도 쉽게 풀 수 있음
- 문자열을 다루는 문제도 많이 나옴
    - 그래서 문자열을 편하게 다룰 수 있는 파이썬 같은 언어가 유리함
