## 투 포인터 / 슬라이딩 윈도우
- 투 포인터와 슬라이딩 윈도우는 비슷한 개념의 알고리즘인데 꽤나 빈번하게 출제됨
- 시간복잡도 O(N^2)나 O(NlogN)으로 구현하면 예시로 주는 테스트케이스는 통과하는데 효율성에서 터져버리는 함정 문제로 많이 출제되는 것 같음
- 개념만 이해하고 있다면 구현도 어렵지 않으니 공부하길 추천
